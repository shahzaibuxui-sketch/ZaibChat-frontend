ZaibChat 🤖
An AI-powered chatbot built with React and Python Flask, connected to Google Gemini 2.5 Flash API.
🌟 Features

💬 Real-time AI chat powered by Google Gemini 2.5 Flash
🌙 Dark / Light mode with localStorage persistence
📝 Chat history with rename and delete support
🎤 Voice input using Web Speech API
📎 File and image upload support
⌨️ Typing indicator and auto-scroll
📱 Fully responsive for mobile and desktop

🛠️ Tech Stack
Frontend: React, Vite, CSS Variables

Backend: Python, Flask, Flask-CORS

AI: Google Gemini 2.5 Flash API

Deployment: Netlify (frontend) + Render (backend)
🚀 Getting Started
Frontend
bashcd ZaibChat-frontend
npm install
npm run dev
Backend
bashcd ZaibChat-backend
pip install -r requirements.txt
python app.py
🔑 Environment Variables
Frontend .env:
VITE_API_URL=http://127.0.0.1:5000
Backend .env:
GEMINI_API_KEY=your_gemini_api_key
